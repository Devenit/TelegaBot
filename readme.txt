Telegram-bot по id @hrizolight_bot
Позволяет вам смотреть товары по трем категорияим: смартфоны, ноутбуки, планшеты в WildBerries.

Бот использует WildBerries Api и предоставляет ссылку на товар, показывает его название, обычную цену и со скидкой.
Используйте команду /start и следуйте инструкциям бота, предварительно запустив main3.py
